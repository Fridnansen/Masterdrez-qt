# 📖 Description

Masterdrez-Qt is a chess-based application developed using Qt and C++ implementing game board logic, piece movement, and chess rule validation. The system is designed as a modular architecture allowing future extensions including multiplayer logic and AI.

The project focuses on building a reusable chess engine architecture for experimentation and research in game development.
